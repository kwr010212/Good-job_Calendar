# 대학생을 위한 일정관리 프로그램(수정)
## 1조 잘하조
팀원 : 정예린 장지영 김시현 노은지 김우림 최은호 곽건우
## 프로젝트 개요
- 대학생들이 학업, 과제, 시험, 팀플, 개인 일정 등을 효율적으로 관리할 수 있도록 돕는 데 목적을 둔 일정 관리 프로그램을 구성했습니다
- GUI를 기반으로 일정을 등록, 수정, 삭제하고, 캘린더 형식으로 한눈에 확인할 수 있는 기능을 제공합니다
## 역할
- 곽건우 : 저장, 불러오기
- 김시현 : 시간표 기능
- 김우림 : 검색, 필터
- 노은지 : 알림 기능
- 정예린 : 일정 등록, 수정
- 장지영 : 코드 구현(GUI)
- 최인호 : 달력 기능
