## 📌 프로젝트 개요

대학생들이 학업과 일상을 보다 **체계적이고 효율적으로 관리**할 수 있도록 돕는 솔루션입니다. 파이썬 기반의 직관적인 GUI를 통해 복잡한 학사 일정을 한눈에 파악하세요!

- 🗂️ **학업 중심**: 과제, 시험, 팀플 등 대학 생활 맞춤형 카테고리
- 🖥️ **직관적인 UX**: 버튼 클릭만으로 간편한 일정 제어
- 📅 **시각화**: 월별/일별 일정을 캘린더 뷰로 즉시 확인
- 🔔 **스마트 알림**: 중요한 마감 기한을 놓치지 않는 리마인드

## 🧩 주요 기능

| 기능 | 설명 |
| --- | --- |
| **CRUD 관리** | 일정의 등록, 수정, 삭제 기능을 통한 유연한 관리 |
| **캘린더 뷰** | 월간 달력 형식을 통한 시각적 일정 파악 |
| **시간표 시스템** | 고정된 강의 시간을 관리하는 전용 시간표 기능 |
| **검색 및 필터** | 원하는 일정만 빠르게 찾을 수 있는 검색 시스템 |
| **데이터 영속성** | 프로그램 종료 후에도 데이터가 유지되는 저장 기능 |
| **알림 서비스** | 임박한 일정을 사용자에게 알려주는 알림창 기능 |
