## 📌 프로젝트 개요

대학생들이 학업과 일상을 보다 **체계적이고 효율적으로 관리**할 수 있도록 돕는 일정 관리 프로그램입니다.

* 🗂️ **학업 중심 일정 관리**: 과제, 시험, 팀플, 개인 일정 통합 관리
* 🖥️ **직관적인 GUI**: 등록 · 수정 · 삭제를 버튼 기반으로 간편하게 수행
* 📅 **캘린더 시각화**: 한눈에 월별/일별 일정 확인 가능
* 🔔 **알림 기능**: 중요한 일정을 놓치지 않도록 사전 알림 제공

---

## 🧩 주요 기능

* 일정 등록 / 수정 / 삭제
* 캘린더 기반 일정 조회
* 시간표 관리 기능
* 일정 검색 및 필터링
* 알림(리마인드) 기능
* 일정 데이터 저장 및 불러오기
